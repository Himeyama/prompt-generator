# PromptGenerator
日本語 | [English](README.en.md) 

![PromptGenerator](docs/image.png)

かんたん画像生成 AI (PromptGenerator) はプロンプト定義ファイルから、UI を構成し簡単に画像生成を可能とするソフトウェアです。

## システム図
![system-structure](docs/structure.drawio.png)

## インストール条件
- Windows 11 インストール済み
- NVIDIA のグラフィックボード搭載端末
- uv コマンドインストール済み
- [genimage](https://github.com/Himeyama/genimage) インストール済み
- モデルファイル準備済み

## 実行条件
- プロンプト定義ファイルの配置
- MCP 設定

## プロンプト定義ファイル
プロンプトは `C:\Users\<ユーザー名>\prompt-generator\prompts.json` に定義します。

### プロンプト設定例
[prompts.json](docs/prompts.json)

## MCP 設定
かんたん画像生成 AI (本ソフトウェア) と [genimage](https://github.com/Himeyama/genimage) でプロセス間通信を MCP で行う仕組みのため、
MCP の設定が必要です。

設定は、`C:\Users\<ユーザー名>\prompt-generator\mcp_config.json` に記載します。

genimage が Windows にインストールされている場合は、以下のように設定を書きます。

```json
{
  "mcpServers": {
    "genimage": {
      "command": "uv",
      "args": [
        "run",
        "--directory",
        "C:/Users/<USERNAME>/genimage",
        "python",
        "-m",
        "main",
        "--model",
        "models/<MODEL>.safetensors",
        "--mcp"
      ]
    }
  }
}
```

## 出力画像について
出力画像は右側に表示されますが、`C:\Users\<ユーザー名>\prompt-generator\images\` にも保存されます。
